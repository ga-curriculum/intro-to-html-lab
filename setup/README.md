## Setup

Open your Terminal application and navigate to your `~/code/ga/labs` directory:

```bash
cd ~/code/ga/labs
```

Make a new repository on [GitHub](https://github.com/) named **intro-to-html-lab**.

Clone a copy of your remote repo locally by using the `git clone` command:

```bash
git clone https://github.com/<your-username>/intro-to-html-lab.git
```

> 📚 Note: In the link above, where it says `<your-username>`, you should see the username from your GitHub account.

Next, `cd` into your new cloned directory, `intro-to-html-lab`:

```bash
cd intro-to-html-lab
```

Open the project's folder in your code editor:

```bash
code .
```

Create a new `index.html` file using the following command: 

```bash
touch index.html
```
